# Welcome to API Whoami

This is a very new (beta) script that aims to take in API keys/tokens and determine which platform they are from, and what permissions may be associated with it.

## Current Support

- [x] Slack API tokens

## Planned Support

- [ ] Facebook
- [ ] Twitter
- [ ] Amazon Web Services
- [ ] Google Cloud Platform
- [ ] MS Azure
